# E-Commerce Microservices

A modular e-commerce system built with .NET, SQL Server, and Clean Architecture.

## Tech Stack
- .NET
- ASP.NET Core Web API
- SQL Server
- Entity Framework Core

## Architecture
- Clean Architecture
- Microservices (in progress)

## Services (planned)
- Catalog Service
- Inventory Service
- Cart Service
- Order Service
- Payment Service
